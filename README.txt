Desafio Take Blip -

Este repositório contém o código resultante do desafio proposto pela Take Blip no processo seletivo para Pessoa Desenvolvedora.

O código fonte consiste em uma API desenvolvida na linguagem C#, utilizando a framework .Net Core versão 3.1 e hospedada em uma função
Lambda hospedada na plataforma online Amazon Web Services.

A API DesafioTakeAPI tem como função recuperar dados dos cinco repositórios de linguagem C# mais antigos disponibilizados pela Take Blip na 
plataforma GitHub, tratar as informações com filtros para exibir o nome do repositório, descrição, data de criação, ícone de usuário e a 
linguagem de programação utilizada. Por fim, a API deve providenciar tais dados para consumo em chatbots, especificamente o chatbot criado
no âmbito do Desafio Take Blip.
